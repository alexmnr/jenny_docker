# Dependencies
To run this docker container, you need docker (duh)

Here you can find instructions on how: https://docs.docker.com/engine/install/

# Build
To build the container:
```sh
./rosco build
```
# Run
To build the container:
```sh
./rosco run
```
# Remove
To remove the container and clean up
```sh
./rosco remove
```
